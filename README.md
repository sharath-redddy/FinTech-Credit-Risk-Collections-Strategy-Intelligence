# FinTech Credit Risk & Collections Strategy Intelligence
Advanced end-to-end credit risk analytics and collections strategy project for a fictional Indian digital lender, CrediSphere Finance.
This project analyzes:
- Loan applications
- Approval and rejection logic
- Repayment behavior
- Delinquency movement
- Default risk
- Expected credit loss
- Collections prioritization
- Credit policy simulation
- Portfolio health
Current project status:
- Workspace created
- Documentation started
- Synthetic raw data generation scripts created
- Raw datasets generated locally
- Raw data validation completed
- Cleaning pipeline created
- Processed datasets created locally
Note:
Generated CSV datasets are excluded from GitHub using .gitignore. They can be regenerated by running the Python scripts.
