# Project Design Direction
## Project Title
FinTech Credit Risk & Collections Strategy Intelligence
## Fictional Company
CrediSphere Finance
## Design Goal
This project must look different from previous portfolio projects. The final Power BI dashboard and Streamlit app should use a bright, premium fintech risk-intelligence design instead of a dark dashboard style.
## Visual Style
- Bright executive fintech theme
- Clean white or soft-light background
- Premium blue, teal, emerald, amber, and red risk accents
- Clear spacing and strong layout hierarchy
- Polished KPI cards
- Modern control-tower layout
- Business-first storytelling
- No crowded chart pages
- No student-style generic dashboard design
- No dark theme as the primary dashboard style
## Dashboard Personality
The dashboard should feel like a real lending-risk leadership product used by:
- Chief Risk Officer
- Credit Risk Manager
- Collections Head
- Portfolio Analytics Team
- Lending Strategy Team
## Color Logic
- Healthy / low risk: Emerald or green
- Watch / medium risk: Amber
- High risk: Orange
- Critical / default / write-off: Red
- Neutral portfolio metrics: Blue or slate
- Premium fintech accents: Teal and cyan
## Differentiation From Previous Projects
This project should not look like:
- SaaS churn dashboard
- E-commerce forecasting dashboard
- Customer segmentation dashboard
- Product growth experimentation dashboard
This project should focus on:
- Lending risk
- Delinquency movement
- Roll-rate intelligence
- Expected credit loss
- Collections prioritization
- Credit policy simulation
- Responsible lending checks
- Portfolio health monitoring
## Final Presentation Standard
The final output should feel recruiter-level and suitable for a strong Risk Data Analyst, Credit Risk Analyst, BI Analyst, or Analytics Engineer portfolio.
